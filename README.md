## ⚙ How It Works

mcp-video-inspector uses a few components working behind the scenes:

- **OpenCV** handles the low-level video processing like reading frames and detecting changes.

- **SceneDetect** scans the video to find where one scene ends and another begins. This helps locate cuts and transitions.

- **AI Language Model (LLM)** explains what each detected feature means. It converts technical data into user-friendly descriptions.

The app combines these tools to give you a clear picture of the video editing without needing you to learn complex software.
